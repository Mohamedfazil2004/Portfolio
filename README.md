# Premium Portfolio Website

This is a recruiter-focused personal portfolio website for Mohamed Fazil G, built with React.js and Framer Motion.

## 🚀 Features

- **Premium Design**: Dark theme with neon cyan and purple accents.
- **Smooth Animations**: Section reveals, hover effects, and page transitions using Framer Motion.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Interactive**: Sticky navbar, tabbed experience section, and custom project cards.
- **SEO Ready**: Proper meta tags and semantic HTML.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Styling**: Vanilla CSS (Global & Component-based)
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Build Tool**: Vite

## 🏃‍♂️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone <your-repo-link>
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
